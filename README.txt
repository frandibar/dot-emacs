INSTALLATION
------------

Plugins must be installed first, for that purpose I use Cask.

Install Cask on OS X using brew:
$ brew install cask

Install Emacs plugins:
$ cask install

Install python extensions:
$ pip install flake8
$ pip install ipython
$ pip install jedi
$ pip install elpy
$ pip install pep8
$ pip install rope

Now run Emacs and enjoy!
